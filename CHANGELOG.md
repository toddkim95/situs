## 0.1.0

- Initial local development version.
- zsh integration with `Ctrl-G`.
- Inline and fullscreen picker modes.
- Situs local history recording.
- Atuin import and optional read-only auto-sync.
- Directory staging with `Tab` and command execution with `Enter`.
